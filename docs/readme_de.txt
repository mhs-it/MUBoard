INSTALLATIONSANLEITUNG
----------------------
1) Kopieren Sie MUBoard in das Verzeichnis modules.
2) Kopiere den Ordner `userdata/MUBoard/` im Ordner `Recource` nach `userdata/MUBoard/`.
3) Setzen Sie das Verzeichnis `userdata/MUBoard/` inklusive aller Unterordner auf beschreibbar.
4) Initialisieren und aktivieren Sie MUBoard in der Modulverwaltung.
